Newcomer Candidates Project

What is this?

This is the result of preliminary survey as part of the ICSME 2020 registered report track "Newcomer Candidate: Characterizing Contributions of a Novice Developer to GitHub". We used the "first-contributions" project and its community in GitHub as our data source for collecting newcomer candidates. Out of the 10,000 newcomer candidates that we contacted, 208 newcomer candidates agreed to let us use their GitHub information and verified that whether they have or not contributed to any OSS project in the past. Below table details the questions asked for the verification of newcomer candidate status.

Survey Questions for Newcomer candidate
Q1) What are their motivation to make a contribution to GitHub?
a. Learning to Code. b. Assignment or Experiment Project. c. Intend to contribute to an Open Source. d. Use to showcase my programming skills. e. Others.
Q2) Did they have any prior experience contributing to an OSS before GitHub? (Yes/No)
Q3) List their programming knowledge/interests?
